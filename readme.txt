WPForms addon for WP Security Audit Log plugin for Wordpress
 
== Description ==
 
An addon for WP Security Audit Log which allows you to see when forms created via WPForms are created, modified and more.

== Installation ==
 
1. Ensure you have the WP Security Audit Log plugin installed.
2. Visit ‘Plugins > Add New’
3. Search for ‘WPForms Add-on for WP Security Audit Log’
4. Install and activate the WPForms Add-on for WP Security Audit Log plugin
